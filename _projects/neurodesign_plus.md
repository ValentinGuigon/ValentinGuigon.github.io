description_paragraphs:
  - I led the development of Neurodesign-plus, an open-source Python package for optimizing fMRI designs in complex decision-making experiments. The package extends neurodesign (R. Poldrack's lab) to support task structures where trial order, timing, or condition assignment can be randomized, probabilistic, or path-dependent.
  - Atharv Umap implemented much of the core software under my supervision. I defined the required functionality, contributed code, guided the package architecture, and am responsible for tutorials and documentation. The package is designed for realistic fMRI experiments where design efficiency must be optimized without oversimplifying the task structure.
